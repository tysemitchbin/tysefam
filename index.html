      📴 Sharing isn't on yet

      Right now this saves only to this browser. To let you both see each other's edits from any device, turn on free sync (about 5 minutes, once):

      1. Go to console.firebase.google.com and create a free project.
      2. Add a Web app (the </> icon). Firebase shows you a firebaseConfig block — copy it.
      3. In the menu, open Build → Realtime Database and create one (any region; start in test mode).
      4. Open index.html in a text editor, find FIREBASE_CONFIG near the top of the script, and paste your values in.
      5. Re-upload to GitHub. The badge turns green and you're synced! 💚

      Full step-by-step is in the README file. Firebase's web keys are safe to keep in a public repo.
